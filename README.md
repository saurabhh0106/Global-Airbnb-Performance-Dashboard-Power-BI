# Global-Airbnb-Performance-Dashboard-Power-BI
This project presents a comprehensive global analysis of Airbnb performance using Power BI, focusing on pricing trends, seasonal demand, customer behavior, and host performance.
The dashboard is designed to simulate real-world business decision-making, helping stakeholders optimize pricing, improve occupancy rates, and identify high-performing markets.

Key Features & Insights
Revenue & Pricing Analysis
Average price distribution across countries and cities
Identification of premium vs budget markets
Price vs occupancy relationship

Seasonality & Trends
Monthly booking trends showing peak seasons
Seasonal pricing fluctuations
Impact of holidays and tourism cycles

Tools & Technologies
Power BI – Dashboard creation & visualization
DAX (Data Analysis Expressions) – Advanced calculations
Power Query – Data transformation & cleaning
Excel / CSV Dataset – Data source

Dataset Source :- https://app.mavenanalytics.io/datasets?search=airbn
